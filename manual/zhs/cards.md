# 卡牌

浏览、筛选并编辑卡牌。对局内可切换牌库 / 手牌 / 抽牌堆 / 弃牌堆 / 消耗堆 / 整副牌；**全部卡牌**标签页浏览完整牌库。

## 筛选芯片

顶部 **类型 / 稀有度 / 费用** 与 **角色 / Special** 池筛选均支持三态：

- **左键（蓝色）**：**包含** — 只显示匹配项（同组多选为 OR）
- **右键（红色）**：**排除** — 隐藏匹配项
- **再右键 / 取消左键（灰色）**：该标签不参与筛选

- **Character** 行：五官方角色 + **Mod 角色 ▾** 下拉（左键勾选包含，右键排除单个 mod 角色）
- **Special** 行：无色、先古、状态、诅咒、事件、任务、代币
- **Library** 行（仅全部卡牌）：**查看升级**、**显示隐藏**（设置项，无排除态）
- 筛选与搜索在对局间**保持**，关闭浏览器后仍生效

## 排序与搜索

- 点击 **Rarity / Type / Cost / A-Z** 切换该维度升序或降序；多维度按优先级链式排序
- 搜索框匹配卡牌名称与描述文本

## 右侧面板

选中卡牌后：

- **对局来源**：加入手牌 / 抽牌堆 / 弃牌堆 / 消耗堆，或从当前区域移除
- **全部卡牌**：预览与编辑（费用、伤害、格挡、消耗、虚无等）、附魔选择、复制 ID
- 网格点击选中；部分模式支持拖拽式操作（与当前标签页一致）
