# Settings

- Game speed, skip animations, UI **theme**
- **Rail layout**: drag to reorder, hide tabs, reset defaults
- Global options such as show hidden cards (synced with the card browser Library row)
- **Progress protection**: auto-backup `progress.save` when the loaded mod set changes; optional warn-only log for removed-mod residue in progress. From the title screen **DEVMODE → Progress protection**, you can restore a backup (main menu only; match the backup mod set when possible).
