# Cards

Browse, filter, and edit cards. In-run tabs: library / hand / draw / discard / exhaust / full deck. **All Cards** browses the full card pool.

## Filter chips

**Type / Rarity / Cost** and **Character / Special** pool filters are tri-state:

- **Left-click (blue)**: **Include** — show matches (OR within a group)
- **Right-click (red)**: **Exclude** — hide matches
- **Right-click again / off (grey)**: tag ignored

- **Character** row: five official characters + **Mods ▾** (left-click include, right-click exclude per mod)
- **Special** row: Colorless, Ancients, Status, Curse, Event, Quest, Token
- **Library** row (All Cards only): **View upgrades**, **Show hidden** (settings toggles, no exclude)
- Filters and search **persist** across browser sessions

## Sort and search

- Click **Rarity / Type / Cost / A-Z** to toggle asc/desc; multiple sort keys chain by priority
- Search matches card name and description text

## Right panel

When a card is selected:

- **In-run sources**: add to hand / draw / discard / exhaust, or remove from current zone
- **All Cards**: preview and edit stats (cost, damage, block, exhaust, ethereal, …), enchantment picker, copy ID
