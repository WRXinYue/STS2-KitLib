All-in-one in-game toolkit for Slay the Spire 2 — test builds, cheat, script, and debug mods without leaving the game.

[img]https://raw.githubusercontent.com/WRXinYue/STS2-DevMode/main/assets/devmode.png[/img]

[b]Panels[/b]

[list]
[*][b]Cheats[/b] — God mode, infinite energy/block, damage multipliers, enemy freeze, stat locks, map overrides, reward tweaks — all gameplay modifiers in one place
[*][b]Card Browser[/b] — Browse the full card library, filter by type/rarity/cost/pool/character, edit cost/damage/block, add to any pile (Hand, Draw, Discard, Deck, Exhaust), view upgrade previews; filters and search remembered across sessions
[*][b]Relic Browser[/b] — Browse and add relics
[*][b]Power Browser[/b] — Browse and apply powers with 4 target modes (self, all enemies, specific, allies); one-click Auto-Apply hook creation
[*][b]Potion Browser[/b] — Visual grid browser; one-click Auto-Apply hook creation
[*][b]Enemy Encounter[/b] — Replace any encounter by room or map node; preview encounter content per node; idle animation preview
[*][b]Hooks[/b] — Define automated Trigger → Condition → Action rules (e.g. add a card every combat start, apply a power on draw)
[*][b]Scripts[/b] — SpireScratch visual block scripting (Blockly); scripts live-reload via WebSocket without restarting the game
[*][b]Save Manager[/b] — Named save slots with dynamic add/delete; carry cards/relics/gold into a new seed run; full slot detail view
[*][b]Console[/b] — Searchable reference for all native and DevMode console commands
[*][b]Log Viewer[/b] — In-game log stream with configurable noise-filter rules
[*][b]Mod Feedback[/b] — Export a ZIP bug report (filtered logs, mod list, Harmony dump, framework snapshot) for any mod author; privacy mode strips user paths
[*][b]Harmony Analysis[/b] — Inspect all active Harmony patches; filter by owner ID; smart summary layout
[*][b]Settings[/b] — Appearance theme (Dark / OLED / Light / Warm), game speed, skip animations
[/list]

《杀戮尖塔 2》全功能游戏内工具箱：测试、作弊、脚本与 Mod 调试一体化。

[b]面板一览[/b]

[list]
[*][b]作弊[/b] — 无敌、无限能量/格挡、伤害倍率、冻结敌人、数值锁定、地图覆盖、奖励调整 — 所有玩法修改项集中一处
[*][b]卡牌浏览器[/b] — 浏览全卡库，按类型/稀有度/费用/卡池/角色筛选，编辑费用/伤害/格挡，添加至任意牌堆（手牌/抽牌/弃牌/牌组/消耗），查看升级对比；筛选与搜索条件跨会话记忆
[*][b]遗物浏览器[/b] — 浏览并添加遗物
[*][b]能力浏览器[/b] — 浏览并施加能力，支持 4 种目标模式（自身、所有敌人、指定、友军）；一键创建「战斗开始自动施加」钩子
[*][b]药水浏览器[/b] — 图标网格展示；一键创建「战斗开始自动使用」钩子
[*][b]敌人遭遇[/b] — 按房间或地图节点替换遭遇，预览各节点遭遇内容；支持待机动画预览
[*][b]钩子[/b] — 定义自动化「触发器 → 条件 → 动作」规则（例如战斗开始添加卡牌、每回合开始施加能力）
[*][b]脚本[/b] — SpireScratch 可视化积木脚本（Blockly）；通过 WebSocket 实时热重载，无需重启游戏
[*][b]存档管理[/b] — 命名存档槽，支持动态增删；可携带卡牌/遗物/金币开新种子局；完整存档详情视图
[*][b]控制台[/b] — 全部原版及 DevMode 控制台命令的可搜索参考手册
[*][b]日志查看器[/b] — 游戏内日志流，可配置噪音过滤规则
[*][b]Mod 反馈[/b] — 为任意 Mod 作者导出 ZIP 问题报告包（降噪日志、Mod 列表、Harmony 转储、框架快照）；隐私模式自动抹去用户路径
[*][b]Harmony 分析[/b] — 查看所有激活的 Harmony 补丁，按 owner ID 筛选，智能摘要排版
[*][b]设置[/b] — 外观主题（Dark / OLED / Light / Warm）、游戏速度、跳过动画
[/list]
